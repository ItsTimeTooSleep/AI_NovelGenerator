# 第一个项目

本教程将带领您完成从项目创建到生成第一章的完整流程！

## 项目概述

我们将创建一部名为《异数边疆》的历史穿越小说，讲述一位理科博士穿越到唐朝的故事。

## 开始之前

确保您已：
- ✅ 完成 [安装指南](installation.md)
- ✅ 配置好 API Key（见 [快速上手](quick-start.md)）

---

## 步骤 1: 创建项目

### 1.1 启动应用

```bash
python.exe main.py
```

### 1.2 新建项目

1. 点击左侧「图书馆」标签
2. 点击「新建项目」按钮
3. 填写以下信息：

| 字段 | 值 |
|------|-----|
| **书名** | 《异数边疆》 |
| **类型** | 历史/爆笑/通俗 |
| **主题** | 理科博士穿到唐朝，被贬边疆，用现代知识改变命运 |
| **计划章节数** | 300 |
| **单章目标字数** | 2500 |

4. 点击「创建」

---

## 步骤 2: 生成小说架构

### 2.1 补充设定

项目创建后自动切换到「项目设定」页面，您可以在编辑器中添加更多细节：

```
世界观补充：
- 时代：唐朝架空，灵晶污染背景
- 地点：寒州边疆，灵蚀河畔
- 核心冲突：现代科学 vs 古代玄幻

主角补充：
- 萧启明，28岁，材料学博士
- 性格：理性、务实、有担当
- 金手指：超越时代的知识体系
```

### 2.2 触发生成

点击「生成架构」按钮，等待 LLM 生成完整架构。

生成结果将包含：
- 完整世界观设定
- 主要角色详细设定
- 主线剧情大纲（分卷）
- 暗线与伏笔设计

### 2.3 查看和编辑

生成完成后，您可以：
- 在编辑器中查看完整内容
- 修改不满意的部分
- 点击「保存」确认修改

> 详细说明请参考 [第一步：生成小说架构](../user-guide/workflow/step1-architecture.md)

---

## 步骤 3: 生成章节目录

### 3.1 切换到目录页面

点击左侧「目录」标签。

### 3.2 确认参数

检查以下设置：
- 总章节数：300
- 分卷设计：启用（建议每 30-50 章一卷）
- 节奏要求：标准

### 3.3 生成目录

点击「生成目录」按钮。

生成的目录将包含每章的：
- 章节标题
- 章节定位（如「主角穿越/世界观引入」）
- 核心作用（如「开局引爆」）
- 悬念密度（爆发/紧凑/渐进）
- 伏笔操作（埋设/回收/强化）
- 详细章节简述

### 3.4 调整目录（可选）

您可以：
- 修改章节标题
- 调整章节顺序
- 增删伏笔设计
- 手动编辑 Markdown 内容

完成后点击「保存」。

> 详细说明请参考 [第二步：生成章节目录](../user-guide/workflow/step2-directory.md)

---

## 步骤 4: 生成第一章

### 4.1 切换到生成台

点击左侧「首页」标签。

### 4.2 选择章节

确保选择的是 **第 1 章**。

### 4.3 填写可选参数（推荐）

在「生成参数」区域填写：

| 字段 | 示例值 |
|------|--------|
| 用户特别指导 | 重点描写主角醒来的迷茫和环境的陌生感 |
| 本章涉及角色 | 萧启明、苏怀瑾 |
| 关键道具 | 破损皇子服、草纸册 |
| 场景地点 | 寒州戍所，病榻 |

### 4.4 生成草稿

点击「生成草稿」按钮。

系统将：
1. 读取架构、目录等依赖文件
2. 构建超长提示词
3. 调用 LLM 生成章节内容
4. 在编辑器中展示结果

### 4.5 编辑和完善

在编辑器中：
- 修改不满意的段落
- 补充细节描写
- 调整对话
- 修正逻辑问题

编辑器支持：
- `Ctrl + Z` 撤销
- `Ctrl + Y` 重做
- `Ctrl + F` 搜索

### 4.6 定稿章节

对内容满意后：
1. 可勾选「运行一致性检查」（推荐）
2. 点击「定稿章节」
3. 等待系统更新：
   - global_summary.txt
   - character_state.txt
   - 向量数据库
   - project.json

> 详细说明请参考 [第三步：生成章节草稿](../user-guide/workflow/step3-generation.md) 和 [第四步：定稿与状态更新](../user-guide/workflow/step4-finalization.md)

---

## 步骤 5: 继续创作

第一章完成后，您可以：
- 继续生成第 2、3... 章
- 在「写作」标签页查看和编辑所有章节
- 在「角色」标签页查看和管理角色状态
- 使用「一致性检查」功能排查问题

---

## 项目文件结构

您的项目文件保存在 `novels/异数边疆/` 目录下：

```
novels/异数边疆/
├── project.json              # 项目配置
├── Novel_architecture.txt    # 小说架构
├── Novel_directory.txt       # 章节目录
├── character_state.txt       # 角色状态
├── global_summary.txt        # 全局摘要
├── cover.png                 # 封面（如有）
└── chapters/
    ├── chapter_1.txt
    ├── chapter_2.txt
    └── ...
```
