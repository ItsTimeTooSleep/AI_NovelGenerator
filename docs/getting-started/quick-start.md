# 快速上手

本指南将帮助您在 5 分钟内开始使用 AI_NovelGenerator 创建您的第一个小说项目！

## 前置条件

在开始之前，请确保您已完成以下准备：

- ✅ Python 3.9 或更高版本（推荐 3.10-3.12）
- ✅ 有效的 LLM API Key（如 OpenAI、DeepSeek 等）
- ✅ 已阅读 [安装指南](installation.md)

## 第一步：启动应用

```bash
python.exe main.py
```

## 第二步：配置 API

1. 点击左侧导航栏的「设置」标签
2. 在「LLM 配置」区域输入您的 API Key
3. 选择或填写 Base URL 和 Model Name
4. 点击「测试连接」确认配置正确

> 详细配置说明请参考 [配置项详解](../reference/config.md)

## 第三步：创建新项目

1. 点击左侧导航栏的「图书馆」标签
2. 点击「新建项目」按钮
3. 填写项目信息：
   - **书名**: 您的小说名称
   - **类型**: 玄幻、科幻、历史等
   - **主题**: 简要描述小说核心设定
   - **计划章节数**: 如 100 章
   - **单章目标字数**: 如 2500 字
4. 点击「创建」

## 第四步：生成小说架构

1. 项目创建后自动切换到「项目设定」页面
2. 您可以在编辑器中补充或修改初始设定
3. 点击「生成架构」按钮
4. 等待 LLM 生成完整的世界观、角色设定和剧情大纲

> 详细流程请参考 [第一步：生成小说架构](../user-guide/workflow/step1-architecture.md)

## 第五步：生成章节目录

1. 切换到「目录」标签页
2. 确认章节参数设置
3. 点击「生成目录」
4. 系统将为您生成完整的章节目录，包含每章定位和伏笔设计

> 详细流程请参考 [第二步：生成章节目录](../user-guide/workflow/step2-directory.md)

## 第六步：开始写第一章

1. 切换到「首页（生成台）」
2. 选择第 1 章
3. 点击「生成草稿」
4. 等待章节生成完成
5. 在编辑器中修改和完善内容
6. 满意后点击「定稿章节」

> 详细流程请参考 [第三步：生成章节草稿](../user-guide/workflow/step3-generation.md) 和 [第四步：定稿与状态更新](../user-guide/workflow/step4-finalization.md)
