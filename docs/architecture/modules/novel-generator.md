# 小说生成器模块

## 概述

本文档详细介绍 `novel_generator/` 目录下的核心模块。

## 模块列表

### composer.py - Composer AI写作助手

Composer是智能写作辅助模块，为编辑器提供语法修正、文本润色、内容扩写、问题查询等AI辅助功能。

#### 核心类

| 类名 | 描述 |
|------|------|
| `ComposerPromptBuilder` | 提示词构建器，根据AI等级构建不同复杂度的提示词 |
| `ComposerAIService` | AI服务封装，处理AI请求和响应 |

#### AI等级说明

| 等级 | 描述 | 适用场景 |
|------|------|----------|
| `mini` | 基础模式，仅处理当前文本片段 | 快速响应，简单任务 |
| `standard` | 标准模式，包含章节上下文信息 | 平衡质量和速度 |
| `pro` | 专业模式，整合完整故事框架、蓝图及角色状态 | 高质量输出 |

#### 支持的任务类型

| 任务类型 | 描述 | 提示词特点 |
|----------|------|------------|
| `grammar` | 语法修正 | 检查句法、用词、标点、逻辑、冗余等问题 |
| `polish` | 文本润色 | 从语言、叙事、风格三个维度优化表达 |
| `expand` | 内容扩写 | 支持5种描写类型，每种有专门的写作技巧指导 |
| `query` | 问题查询 | 提供专业的创作顾问服务 |

#### 扩展描写类型

| 类型ID | 中文名称 | 描写重点 |
|--------|----------|----------|
| `psychological` | 心理描写 | 内心独白、情感层次、潜意识流露、矛盾冲突 |
| `expression` | 神态描写 | 眼神、微表情、面部肌肉、神态与情绪 |
| `action` | 动作描写 | 动作分解、力量节奏、动作特征、环境互动 |
| `environment` | 环境描写 | 感官细节、空间层次、光影效果、氛围营造 |
| `dialogue` | 对话补充 | 语言风格、潜台词、对话节奏、对话张力 |

#### 设计决策

- **分层提示词设计**：适应不同使用场景和Token消耗需求
- **上下文信息按需加载**：控制Token消耗，提高响应效率
- **支持自定义LLM适配器**：兼容多种模型
- **专业化的写作指导**：每种任务类型都有详细的技巧和原则说明
